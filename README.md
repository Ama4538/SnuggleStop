## Installation
1. **Clone the repository:**
    ```bash
    git clone []
    cd SnuggleStop
    ```
2. **Install dependencies:**
    ```bash
    npm i
    ```
3. **Start the server:**
    ```bash
    npm start
    ```
4. **Access the application:**
Open your browser and go to http://localhost:8080.
